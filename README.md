# Takansvar Website Template

This is a simple one-page website template, customized for Takansvar AS. It is suitable for company, personal, or portfolio pages.

## Features
- Responsive design using Bootstrap
- Service steps and guarantee sections
- Contact form for free roof inspection ("Bestill gratis taksjekk")
- Easy to customize with your own content and images

## Usage
1. Download or clone this repository
2. Open `index.html` and edit the content to fit your company details
3. Add your logo and update contact information in the footer
4. Replace placeholder images with your own in the `img/` folder
5. (Optional) Move custom JavaScript from the HTML file to a separate file in the `js/` folder for better maintainability

## Folder Structure
- `index.html` – Main website file
- `css/` – Stylesheets
- `js/` – JavaScript files
- `img/` – Images and portfolio images
- `vendor/` – Bootstrap, Font Awesome, and jQuery libraries
