# Takansvar Website

A professional one-page website template for Takansvar AS, built with Bootstrap for responsive design across all devices.

## Features

- Fully responsive design
- Service steps and company guarantee sections
- Contact form for free roof inspection requests
- Portfolio section for project showcase
- Clean, professional styling
- Easy to customize and extend

## Getting Started

1. Open `index.html` in your browser to view the site
2. Edit content directly in `index.html` to update company information
3. Replace placeholder images in the `img/` folder with your own
4. Update contact information and links in the footer section
5. Customize colors and styling in `css/freelancer.css`

## Project Structure

```
├── index.html              Main website file
├── index.js                Custom JavaScript
├── css/                    Stylesheets
├── js/                     JavaScript files
├── img/                    Images and portfolio content
└── vendor/                 Dependencies (Bootstrap, Font Awesome, jQuery)
```

## Customization

- **Content:** Edit text, headings, and sections in `index.html`
- **Styling:** Modify colors and fonts in `css/freelancer.css`
- **Images:** Replace placeholder images in `img/` with your own
- **JavaScript:** Add custom functionality in `js/contact_me.js`

## Dependencies

- Bootstrap 3 - Responsive framework
- Font Awesome - Icon library
- jQuery - JavaScript library
